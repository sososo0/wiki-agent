이 페이지는 자가 갱신형 위키의 현재 상태를 그래프로 보여줍니다. 각 점(노드)은 위키 항목 하나이고, 항목 간 선(엣지)은 "이 항목이 다른 항목을 대체하려는 후보다" 같은 구조적 관계나 "두 항목이 의미적으로 비슷하다"는 관계를 나타냅니다. 이를 통해 AI가 어떤 항목을 새로 만들었는지, 무엇을 승격 대기 중인지, 안전장치(게이트)가 무엇을 막았는지를 한눈에 볼 수 있습니다.

노드 상태 (위키 항목)

active — 지금 검색/답변에 실제로 쓰이는 항목
shadow — AI가 만든 후보, 아직 승격되지 않고 대기 중
deprecated — 과거엔 active였지만 다른 항목으로 대체되어 강등된 항목
rejected — 안전장치(게이트)가 신뢰할 수 없다고 판단해 막은 후보

엣지 종류 (항목 간 관계)

pending_update — shadow 항목이 대체하려는 active 항목을 가리킴(승격 대기)
superseded_by — deprecated 항목이 과거에 대체했던 대상을 가리킴(교체 이력)
similar — 두 항목의 내용이 임베딩 기준 서로 비슷함(방향 없음, 선이 두꺼울수록 더 비슷함)

노드를 클릭하면 상세 정보가 표시됩니다.
