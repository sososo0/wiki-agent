자가 갱신형 위키 기반 RAG 어시스턴트

이 에이전트는 무엇인가요?

이 에이전트는 사내 위키(지식베이스)에서 질문과 관련된 항목을 검색하고, 검색된 내용만 근거로 답변합니다. 답변에 쓰인 위키 항목은 매 턴 기록되고, 사용자가 남기는 👍/👎 피드백은 위키를 더 정확하게 자동으로 갱신하는 데 쓰입니다(AI가 직접 위키를 고치는 게 아니라, 안전장치를 통과한 후보만 반영됩니다).

사용 방법

  • 알고 싶은 내용을 자연어 질문으로 입력하세요.
  • 답변 아래 "근거"에 실제로 참고한 위키 항목 ID가 표시됩니다.
  • 답변이 만족스러우면 👍, 아니면 👎를 눌러주세요 — 이 피드백이 위키 개선에 직접 쓰입니다.
  • 위키 항목들이 어떻게 연결되어 있는지 보고 싶다면 아래 "그래프 보기"를 확인하세요.
📊 그래프 보기

위키 항목들이 서로 어떻게 연결되어 있는지, 지금 승격 대기 중인 후보는 무엇인지, AI 안전장치(게이트)가 어떤 항목을 막았는지를 시각적으로 보여줍니다.
