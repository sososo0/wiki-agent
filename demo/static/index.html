API/백엔드 요청 신뢰성 패턴 위키 어시스턴트

이 에이전트는 무엇인가요?

이 에이전트는 API/백엔드 요청 신뢰성 패턴을 주로 다루는 위키(지식베이스)에서 질문과 관련된 항목을 검색하고, 검색된 내용만 근거로 답변합니다. 답변에 쓰인 위키 항목은 매 턴 기록되고, 사용자가 남기는 👍/👎 피드백은 위키를 더 정확하게 자동으로 갱신하는 데 쓰입니다(AI가 직접 위키를 고치는 게 아니라, 안전장치를 통과한 후보만 반영됩니다).

재시도/백오프레이트 리미팅서킷 브레이커 커넥션 풀링멱등성타임아웃 전략 벌크헤드 격리백프레셔

사용 방법

  • 알고 싶은 내용을 자연어 질문으로 입력하세요(위 영역 밖의 질문은 모른다고 답할 수 있습니다).
  • 답변 아래 "근거"에 실제로 참고한 위키 항목 ID가 표시됩니다.
  • 답변이 만족스러우면 👍, 아니면 👎를 눌러주세요 — 이 피드백이 위키 개선에 직접 쓰입니다.
  • 위키 항목들이 어떻게 연결되어 있는지 보고 싶다면 아래 "그래프 보기"를 확인하세요.
